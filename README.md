# common-toolkits

## Usage & Guide

To check out live examples and docs, visit [common-toolkits now](https://common-toolkits.vercel.app/).

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
