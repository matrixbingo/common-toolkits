### Params
| 参数    | 说明           | 类型             | 默认值 | 必填 |
| ------- | -------------- | ---------------- | ------ | ---- |
| num     | 数字或字符数字 | string \| number | -      | 是   |
| percent | 百分率符号     | string           | %      | 否   |
