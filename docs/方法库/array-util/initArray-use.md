## API
```jsx | pure
ArrayUtil.initArray(number, value);
```

### Params

```jsx | pure
<T>(length: number = 1, value: T): Array<T>
```

| 参数   | 说明     | 类型   | 默认值 | 必填 |
| ------ | -------- | ------ | ------ | ---- |
| number | 数组长度 | number | 1      | 是   |
| value  | 默认值   | T      | -      | 否   |
