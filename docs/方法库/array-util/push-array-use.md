## API
```jsx | pure
ArrayUtil.pushArray(arr, list, [customizer]);
```

### Params

| 参数       | 说明       | 类型        | 默认值 | 必填 |
| ---------- | ---------- | ----------- | ------ | ---- |
| arr        | 数组       | any[]       | -      | 是   |
| list       | 数组       | any[]       | -      | 是   |
| customizer | 自定义方法 | (i) => true | -      | 否   |
