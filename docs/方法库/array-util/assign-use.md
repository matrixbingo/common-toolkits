## API
```jsx | pure
ArrayUtil.assign(arr, item);
```

### Params

```jsx | pure
<T, S>(arr: T[], item: S): T[]
```
| 参数 | 说明 | 类型 | 默认值 | 必填 |
| ---- | ---- | ---- | ------ | ---- |
| arr  | 数组 | T[]  | -      | 是   |
| item | 对象 | S    | -      | 是   |
